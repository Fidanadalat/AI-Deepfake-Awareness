This repository contains the source code for the AI Deepfake Awareness website, an educational platform designed to raise awareness about deepfakes and promote digital safety. The website covers topics such as deepfake awareness, detection tips, protection policies, and digital safety, providing users with accessible information and interactive resources. The project was developed using HTML, CSS, JavaScript, PHP, and SQL, with a focus on user engagement and data-driven content. The repository is structured to support easy deployment, ongoing updates, and collaboration, making it a valuable resource for anyone interested in combating digital misinformation and enhancing digital literacy.
